# 和訳スキル（英語 → 日本語）

## 入力

`$ARGUMENTS`

## 手順

1. **入力の解釈**
   - ファイルパスが渡された場合: Readツールでファイル内容を読み取り、その全文を翻訳対象とする
   - URLが渡された場合: WebFetchで内容を取得し、その全文を翻訳対象とする
   - それ以外: 渡されたテキストをそのまま翻訳対象とする

2. **翻訳の出力**
   - 日本語訳を **1パターン** 出力する
   - 自然で読みやすい日本語にする
   - 必要に応じて補足説明を英語で簡潔に添える

## ルール

- 技術用語、コード識別子、固有名詞はそのまま維持する
- 翻訳結果のみを出力し、手順の説明や前置きは不要
